神经网络——一阶优化算法

发表于 2019-11-13 更新于 2019-11-25 分类于 深度学习

1. 简介

1.1 神经网络的优化算法

  对于一个神经网络模型，我们想要获得最优的参数，使得损失函数最小。通常情况下目标损失函数的解析解是没法求出来的，损失函数梯度等于0的方程通常是一个超越方程（含有指数函数，对数函数的方程）。对于这种无法直接求解的方程组，通常采用优化算法求出其数值解。
  数值优化算法通常利用了目标函数的导数信息，如果采用一阶导数，则称为一阶优化算法；如果采用二阶导数，则称为我二阶优化算法。工程上通常采用的是迭代法，从一个初始点x0x_0x0​开始，反复使用某种规则从xkx_kxk​移动到下一个点xk+1x_{k+1}xk+1​，构造这样一个数列，直至收敛或近似收敛到梯度为0的点。即有下面的极限成立：

lim⁡k→+∞∇f(xk)=0\lim_{k \rightarrow +\infin} \nabla{f(x_k)}=0 k→+∞lim​∇f(xk​)=0

1.2 一维梯度搜索算法

  假设连续可到的函数f:R→Rf: \mathbb{R} \rightarrow \mathbb{R}f:R→R的输入都是实数域上的一个标量。给定绝对值足够小的数ϵ=Δx\epsilon = \Delta{x}ϵ=Δx为自变量的变化量，根据泰勒展开公式，得到以下近似：

f(x+ϵ)≈f(x)+ϵf′(x)f(x + \epsilon) \approx f(x) + \epsilon f'(x) f(x+ϵ)≈f(x)+ϵf′(x)

其中，f′(x)f'(x)f′(x)是函数fff在xxx处的梯度。上式可以改写为：

f(x+ϵ)−f(x)≈ϵf′(x)f(x + \epsilon) - f(x) \approx \epsilon f'(x) f(x+ϵ)−f(x)≈ϵf′(x)

继续改写为：

Δf(x)≈ϵ∇f\Delta{f(x)} \approx \epsilon \nabla{f} Δf(x)≈ϵ∇f

梯度下降优化的过程要使得Δf(x)\Delta{f(x)}Δf(x)为负，这样f(x)f(x)f(x)的值才在减小。假设选取：

ϵ=−η∇f\epsilon = - \eta \nabla {f} ϵ=−η∇f

其中，η\etaη是一个很小的正数，称为学习率。则有：

Δf(x)≈−η ∇f⋅∇f=−η ∥∇f∥2\Delta{f(x)} \approx -\eta \ \nabla {f} \cdot \nabla {f} = -\eta \ \lVert \nabla {f} \rVert ^{2} Δf(x)≈−η ∇f⋅∇f=−η ∥∇f∥2

上式可以确保Δf(x)≤0\Delta{f(x)} \leq 0Δf(x)≤0恒成立，即f(x)f(x)f(x)会一直减小。因此，可以通过设定ϵ=−η∇f\epsilon= - \eta \nabla {f}ϵ=−η∇f来迭代xxx，确保函数f(x)f(x)f(x)的值不会上升：

x←x−η∇fx \leftarrow x - \eta \nabla {f} x←x−η∇f

因此在一维搜索算法中，先选取一个初始值x0x_{0}x0​和一个小的正数η\etaη，然后不断通过上式迭代求出新的xxx，直至目标函数收敛，例如函数值的更新非常小或者达到指定的迭代次数。

1.3 学习率的设置

  学习率设置过小时，目标函数在后期的收敛速度会很慢；学习率设置过大时，目标函数可能会发散而不是收敛。

1.4 多维梯度搜索算法

  当目标函数f:Rd→Rf: \mathbb{R}^{d} \rightarrow \mathbb{R}f:Rd→R的输入是一个d维向量x=[x1,x2,...,xd]Tx = [x_1, x_2,...,x_d]^Tx=[x1​,x2​,...,xd​]T，输出为一个标量时，目标函数f(x)f(x)f(x)对自变量的偏导数是一个向量：

∇xf(x)=[∂f(x)∂x1,∂f(x)∂x2,...,∂f(x)∂xd]T\nabla_{x}{f(x)} = [\frac {\partial{f(x)}} {\partial {x_1}}, \frac {\partial{f(x)}} {\partial {x_2}},..., \frac {\partial{f(x)}} {\partial {x_d}}]^T ∇x​f(x)=[∂x1​∂f(x)​,∂x2​∂f(x)​,...,∂xd​∂f(x)​]T

其搜索过程和一维搜索一样。

2. 一阶优化算法

  在深度学习里，目标函数通常是训练数据集中有关各个样本的算是函数的平均，设fi(x)f_i(x)fi​(x)是有关索引为iii的训练数据样本的损失函数，nnn是训练数据样本数，XXX是模型的参数向量，那么目标函数的定义为：

f(x)=1n∑i=1nfi(x)f(x) = \frac 1 n \sum_{i=1}^{n}f_i(x) f(x)=n1​i=1∑n​fi​(x)

目标函数在xxx处的梯度计算公式为：

∇f(x)=1n∑i=1n∇fi(x)\nabla {f(x)} = \frac 1 n \sum_{i=1}^{n} \nabla f_i(x) ∇f(x)=n1​i=1∑n​∇fi​(x)

如果直接使用梯度下降算法，每次迭代的过程中需要对所有样本求梯度，计算开销为O(n)O(n)O(n),它随着nnn线性增长。因此，当训练数据样本数很大时，梯度下降算法每次迭代的计算开销很大。

2.1 随机梯度下降法

  随机梯度下降法(Stotastic Gradient Descend method, SGD)在反向传播每次迭代过程中，随机均匀采样得到一个样本索引i∈{1,2,...,n}i \in \{ 1,2,...,n \}i∈{1,2,...,n}，并计算梯度∇fi(x)\nabla f_i(x)∇fi​(x)来迭代更新权重参数x：

x←x−η∇fixx \leftarrow x - \eta \nabla f_i{x} x←x−η∇fi​x

随机梯度下降算法只对一个样本求梯度，算法时间复杂度从O(n)O(n)O(n)下降到了O(1)O(1)O(1)。值得强调的是，随机梯度∇fi(x)\nabla f_i(x)∇fi​(x)是对梯度∇f(x)\nabla f(x)∇f(x)的无偏估计，即：

E(∇fi(x))=1n∑i=1n∇fi(x)=∇f(x)E(\nabla f_i(x)) = \frac 1 n \sum_{i=1}^{n} \nabla f_i(x) = \nabla {f(x)} E(∇fi​(x))=n1​i=1∑n​∇fi​(x)=∇f(x)

这意味着：平均意义而言，随机梯度是对梯度的一个无偏估计。

2.2 批量梯度下降法

  在每次迭代中，使用整个训练数据集计算梯度，这就是批量梯度下降法(Batch Gradient Descend)。

2.3 小批量随机梯度下降法

  如果我们也可以在每轮迭代时随机均匀采集多个样本组成一个小批量，用这个小批量计算梯度后去更新权重参数。这种方法称为小批量梯度下降法(Mini-Batch Gradient Descend)。其中，小批量样本的个数BBB是一个超参数。计算步骤：对batch_size个样本前向传播计算其损失函数，然后用这批样本的损失函数和梯度去更新权重。
  Mini-Batch的大小：太大的batch会使得训练更快，但是可能导致泛化能力下降…有待补充

2.4 动量梯度下降法

  梯度下降法每次迭代时，自变量沿着当前位置的梯度更新，这样的更新可能会很不稳定。动量梯度下降法(Momentum Gradient Descend)在更新自变量的时候一定程度保留之前的更新方向，同时利用了当前batch的梯度微调更新方向，能够增强优化过程的稳定性，并且能一定程度的摆脱局部最优的能力。
  设时间步为ttt的自变量为xtx_txt​，学习率为ηt\eta _tηt​。在时间步000，动量法创建速度变量v0v_0v0​并将其初始化为零。在时间步ttt，动量法对每次迭代的步骤做如下的修改：

vt←γvt+ηtgtxt←xt−vt\begin{aligned} v_t \leftarrow \gamma v_t + \eta_{t}g_t \\ x_t \leftarrow x_t - v_t \end{aligned}vt​←γvt​+ηt​gt​xt​←xt​−vt​​

其中，动量超参数γ\gammaγ满足0≤γ≤10 \leq \gamma \leq 10≤γ≤1，当γ=0\gamma = 0γ=0时，动量法等价于小批量随机梯度下降法。
  动量梯度下降法经过加权移动平均后，在指向极小值的方向上的速度vvv不断加强，在垂直于极小值的方向上速度vvv不断被抵消。

  上面的方法对所有的参数都使用同样的学习率，但是同样的学习率不一定适合所有的参数。有的参数可能需要微调，有的参数由于对应样本少等原因，可能需要较大幅度的调整。Adagrad算法能够自适应地为不同参数设置不同的学习率。

2.7 RMSprop

  Adagrad算法存在以下问题：学习率是单调递减的，训练后期学习率非常小，可能较难找到一个有用的解。RMSProp算法对状态变量sts_tst​进行指数加权移动平均。

st←γst−1+(1−γ)gt⊙gts_t \leftarrow \gamma s_{t-1} + (1-\gamma)g_t \odot g_t st​←γst−1​+(1−γ)gt​⊙gt​

然后更新自变量xtx_txt​

2.8 Adadelta

  Adagrad算法需要手工设置一个全局的初始学习率，并且在更新参数时，左右两边的单位不统一。而Adadelta除了针对Adagrad在训练后期收敛变慢做了改进，而且没有学习率这一超参数。

st←ρst−1+(1−ρ)gt⊙gts_t \leftarrow \rho s_{t-1} + (1-\rho)g_t \odot g_t st​←ρst−1​+(1−ρ)gt​⊙gt​

此外，Adadelta还维护一个额外的状态变量Δxt\Delta x_tΔxt​，

g′t←Δxt−1+ϵst+ϵ⊙gt{g'}_t \leftarrow \sqrt{\frac {\Delta x_{t-1} + \epsilon} {s_t + \epsilon}} \odot g_t g′t​←st​+ϵΔxt−1​+ϵ​​⊙gt​

xt←xt−1−g′tx_t \leftarrow x_{t-1} - {g'}_t xt​←xt−1​−g′t​

Δxt←ρΔxt−1+(1−ρ)g′t⊙g′t\Delta{x_t} \leftarrow \rho \Delta{x_{t-1}} + (1-\rho){g'}_t \odot {g'}_t Δxt​←ρΔxt−1​+(1−ρ)g′t​⊙g′t​

可以看出，Adadelta跟RMSProp的区别在于使用了Δxt−1\sqrt{\Delta{x_{t-1}}}Δxt−1​​来代替超参数η\etaη。
