1.2 卷积神经网络——边缘检测

1.4 padding

  做padding的目的是避免图片边缘的信息丢失了。
  选择padding的个数，通常有两种选择：valid和same。valid表示没有填充；same表示：“Pad so that output size is the same as the input size.”
  通常情况下，卷积核的尺寸是奇数。如果卷积核尺寸是偶数，stride等于1，pandding为same，这时候只能设置不对称的padding了。

1.5 卷积步长strides

  在卷积神经网络中，矩阵经过卷积操作后的尺寸和输入尺寸、卷积核尺寸、padding、滑动步长strides之间的关系如下所示。注意：⌊x⌋\lfloor x\rfloor⌊x⌋表示对xxx向下取整，这意味着一旦卷积核滑动到不能完全在矩阵内时，不做卷积操作。

output_size=⌊input_size−kernal_size+padding×2stride⌋+1output\_size = \lfloor \frac {input\_size - kernal\_size + padding \times 2} {stride} \rfloor + 1 output_size=⌊strideinput_size−kernal_size+padding×2​⌋+1

  备注：在数学上或信号处理意义上的卷积，是先要把卷积核沿水平轴和垂直轴先进行翻转的，再进行卷积运算。在深度学习里面的卷积操作实际应该被称为互相关操作(cross-correlation)。

1.6 三维卷积

  对于含有多个通道的图像(如三通道的RGB图像)，分别在每个通道上进行卷积操作后，在各个通道上得到的结果相加。

1.8 简单卷积网络示例

  一般随着网络的加深，中间层的尺寸刚开始保持一致，后面不断变小；通道数不断增加。
  卷积神经网络中通常有3种layer：卷积层convolution layer，池化层pooling layer，全连接层fully connected。

1.9 池化层

  池化层没有参数要学习，池化层的输出通道数和输入通道数一样。
  池化层的超参数主要是filter size和stride，二者通常均被设置为2，很少设置padding。
  Max pooling层的作用：如果过滤器filter提取到了某个特征，那么保留其最大值。
  Average polling层：使用较少。

1.10 卷积神经网络示例

  在计算卷积神经网络的层数时有：有人把卷积层和池化层作为一层，也有人把卷积层和池化层分开算作两层。
  吴恩达老师把卷积操作和池化操作合起来当作一层。
  卷积神经网络的activation size（每一层的变量个数）通常是在下降，但不能下降太快。
  选择超参数时，尽量参考别人的文章或者网络结构。

1.11 为什么使用卷积

  卷积层和全连接层相比的优势：通过参数共享和稀疏连接减少参数，防止过拟合。
  Parameter sharing: A feature detector(such as a vertical edge detector) that is useful in one part of the image is probably useful in another part of the image.
  Sparsity of connections: In early layer, each output value depends only on a small number of inputs.
