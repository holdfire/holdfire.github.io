Python——包、模块和import

2019-10-30 编程语言 Python 0 评论 字数统计: 913(字) 阅读时长: 3(分)

1. 简介

1.1 基本定义

  包(package)是一个文件夹，在一个文件夹下面加入__init__.py文件(以下简称init文件)，这个文件夹就成为了一个包。模块(module)是包下面的文件，类是文件的基本单元，类中包括变量(或叫属性, attributes)和函数(或叫方法, method)。

1.2 init文件的作用
  • 在文件夹中加入init文件，文件夹就成了一个Python包；
  • 当这个包或者该包下的模块被导入时，init文件都会被执行；
  • import 包名就可以批量导入该包下面的模块；
  • 在init文件中，导入当前目录下的模块应该使用from . import 模块名；
  • init文件中通常含有一个__all__列表(以下简称all列表)
1.3 all列表的作用
  • 该列表控制的是import *语句引入的内容，可以在init文件或者模块中使用；
  • 该列表的内容可以是本文件自定义的变量或函数，或者是导入进来的模块；
  • 在init文件中设置all列表时，from 包名 import *操作只会导入all列表中的内容；
  • 在模块中设置all列表时，from 模块名 import *只会导入all列表指定的内容，称为内置变量；
  • 内置变量易覆盖本文件中的变量名，因而不建议使用from 模块名 import *

  • 每个包或模块只会被导入一次。
  • 要避免循环导入。
