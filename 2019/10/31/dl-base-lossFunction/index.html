机器学习——损失函数

发表于 2019-10-31 更新于 2019-11-18 分类于 机器学习

1. 损失函数简介

  损失函数(Loss Function)是用来估量模型的预测值f(x)f(x)f(x)和真实值YYY的不一致程度，它是一个非负的的实值函数，通常使用L(Y,f(x))L(Y, f(x))L(Y,f(x))来表示。
  损失函数是经验风险函数和核心部分，也是结构风险函数的重要组成部分，结构风险函数包括经验风险项和正则化项。其中，经验风险越小说明模型对训练集的拟合程度越好。

2.回归问题中的损失函数

2.1 平均绝对损失函数L1 Loss

  平均绝对损失函数(Mean Absolute Loss)衡量的是预测值绝对误差的平均值。使用L1损失函数的模型，在迭代过程中参数更新的梯度大小始终不变。也就是说，即使对于很小的损失值，梯度也很大，这种方式可能会不利于模型的学习。为了解决这个缺陷，可以使用变化的学习率。

2.2 均方损失L2

  L2损失对训练集中的异常点（预测值和真是值差别较大的点）较为敏感

2.3 Huber Loss

  Huber损失函数结合了L1损失函数和L2损失函数的优点，对异常点不如L2损失敏感，且在原点可导。超参数δ\deltaδ的选择很关键。

2.5 Quntile Loss

  当我们想要进行区间预测而不仅是点估计时，可以考虑使用分位数损失函数。最小二乘法进行区间预测，是基于残差是独立变量，且方差保持不变（Gauss-Markov假设）。一旦违背了这条假设，那么线性回归模型就不成立。这时，分位数损失和分位数回归就派上用场了，因为即便对于具有变化方差或非正态分布的残差，基于分位数损失的回归也能给出合理的预测区间。

  如何选取合适的分位值取决于我们对正误差和反误差的重视程度。损失函数通过分位值γ\gammaγ对高估和低估给与不同的惩罚。例如，当分位数损失函数γ=0.25\gamma=0.25γ=0.25时，对高估的惩罚更大，使得预测值略低于中值。如果分别取分位数为0.05和0.95，我们就可以得到两个预测模型，这两个预测模型就是90%的预测区间，其物理意义是训练样本的误差有90%落在这个区间内。

3. 分类问题中的损失函数

3.1 0-1损失函数

  0-1损失函数：当y=f(x)y=f(x)y=f(x)时的损失为0，否则为1。0-1损失函数非凸，不连续，无法求导，不适用于深度学习。其对每个序偶分类点都施以相同的惩罚，对犯错比较大的点无法进行较大的惩罚。
  放宽条件后：当∣y−f(x)∣<threshold|y-f(x)| < threshold∣y−f(x)∣<threshold时的损失为0，否则为1。
