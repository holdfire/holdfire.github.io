学习方法——如何阅读论文

发表于 2019-11-04 更新于 2019-11-15 分类于 个人杂记

1.论文的阅读顺序

第一阶段：判断论文是否值得读

阅读顺序为：读标题和关键词–>读摘要–>读结论
掌握论文的大概内容。

第二阶段：读论文

阅读顺序为：读图表和数据–>读引言introduction
如果想要深挖细节：读结果和讨论–>读实验部分

第三阶段：做笔记

好笔记的标准：下次看笔记内容即可，无需再看这篇论文

2.阅读时应该做的思考和笔记

一篇论文的核心往往就一两句话，而论文的其他部分都是在用不同的方式支持它的核心。

摘要Abstract

  摘要是对全篇内容的概括，阅读后需要做的笔记：

  • 作者想要解决什么问题？question
  • 作者通过什么方法/模型解决了这个问题？method
  • 作者给出的答案是什么？result
引言introduction

  引言介绍了文章的研究背景和研究意义，一般在最后会给出文章的结构。需要做的笔记：

  • 作者为什么要研究这个问题？
  • 这个问题现在的研究进展到了哪个地步？
  • 作者使用的理论是基于哪些假设？
结论conclusion

  结论部分会把整篇文章的主要内容复述一遍，帮助读者回顾+理清思路，然后在此基础上加深自己的研究。需要做的笔记：

  • 这篇文章存在哪些缺陷？
  • 作者关于这个课题的构思有哪几点？

神经网络——参数初始化

发表于 2019-11-02 更新于 2019-11-12 分类于 深度学习

1.如何分析参数初始化结果的好坏？

  查看初始化后各层的激活值分布，如果在某个固定区间内分布则较好，如果集中在某个值上则初始化不好。

2. 把w初始化为0

  对于单层网络可行；对于多层网络，由于链式法则会导致梯度消失。

3. 随机初始化

  使用均值为0，方差为0.02的正态分布去初始化。w初始值较小是因为：如果x很大的话，w又相对较大，会导致Z非常大，这样如果激活函数是sigmoid，就会导致sigmoid的输出值1或者0。

4. Xavier initialization

  Xavier initialization方法是Glorot等人为了解决随机初始化的问题提出来的另一种初始化方法。他们的思想倒也简单，就是尽可能的让输入和输出服从相同的分布，这样就能够避免后面层的激活函数的输出值趋向于0。Xavier initialization能够很好的tanh激活函数。

数据预处理——特征缩放

发表于 2019-11-02 更新于 2019-11-25 分类于 深度学习

1. 特征缩放

1.1 特征缩放的定义(what)

  机器学习中常用的标准化(Standardization)和归一化(Normalization)都是对数据集的某种特征进行缩放，这两个概念经常会被混淆，其内涵究竟是什么呢？根据维基百科的定义，特征缩放(Feature scaling)主要有以下四种方法，其内涵包括标准化和归一化。
Rescaling(min-max normalization):（经常被简称为归一化，其实只是归一化的一种）
  最大-最小值归一化可以将特征xxx缩放到一个固定区间[a,b][a,b][a,b]，把椭圆变成了圆。如公式（1）所示：左式可以把特征缩放到[0,1][0,1][0,1]之间，右式可以把特征缩放到任意区间[a,b][a,b][a,b]。

x′=a+(x−min(x))(b−a)max(x)−min(x)x′=x−min(x)max(x)−min(x)(1)\tag{1} x' = a + \frac {(x - min(x))(b-a)} {max(x) - min(x)} \qquad x' = \frac {x - min(x)} {max(x) - min(x)} x′=a+max(x)−min(x)(x−min(x))(b−a)​x′=max(x)−min(x)x−min(x)​(1)

Mean normalization:

x′=x−mean(x)max(x)−min(x)(2)\tag{2} x' = \frac {x - mean(x)} {max(x) - min(x)} x′=max(x)−min(x)x−mean(x)​(2)

Standardization(Z-score normalization):(标准化，是归一化的一种)
  机器学习使用的数据通常包含较多的特征，Standardization处理可以把每种特征都处理成零均值、单位标准差，Standardization常被用在 support vector machines, logistic regression, and artificial neural networks上，实际使用时需要注意分母的标准差可能为零的情况。

x′=x−mean(x)std(x)(3)\tag{3} x' = \frac {x - mean(x)} {std(x)} x′=std(x)x−mean(x)​(3)

Scaling to unit length:

x′=x∣∣x∣∣(4)\tag{4} x' = \frac x {||x||} x′=∣∣x∣∣x​(4)

  各种博客上常见的错误说法是认为归一化是公式（1），标准化是公式（3），个人认为把nornalization翻译成归一化有一定的误导作用。总的来说：Standardization指公式（3），是Normalization的一种；而Normalization有好几种方式，它们可以统称为特征缩放。特征缩放的本质是对特征进行线性变换，线性变换有很多良好的性质，比如线性变换不会改变原始数据的数值排序。

1.2 为什么要做特征缩放(why)

  加速收敛过程：min-max normalization可以将原来椭圆分布的特征转换成了圆形。未做尺度缩放时，迭代时很可能走之字形路线，收敛速度很慢；缩放后可以加快梯度下降的求解速度，提高了模型的收敛速度。（参考自吴恩达深度学习课程第 课第 周）

  无量纲化：有些分类器需要计算样本之间的距离（如k近邻回归），如果某个特征的值域非常大，那么计算的距离就主要取决于这个特征，这可能和实际情况不符合。此外，有些特征的单位无法换算但同样重要时（如房子的数量和个人收入），归一化处理比较符合业务逻辑。
  数据处理成零均值后，可以避免参数更新时发生zig-zag现象。
  最后，太大的数字会引发数值问题（比如上溢）。

1.3 如何选择特征缩放方法(how)

  归一化的缩放是将各个特征统一到同一个区间，其缩放仅和最大值、最小值的差别有关；标准化的缩放和每个点都有关系，通过方差体现出来。
  如果数据较为稳定，不存在极端的最大值和最小值，可以使用归一化。
  如果数据存在较多异常值或噪音，可以使用标准化，以避免异常值和极端值的影响。

1.4 数据预处理——PCA/白化whitening

  以图像训练为例，由于图像中相邻像素间具有很强的相关性，所以用于训练的输入是冗余的。白化的目的就是降低输入的冗余性，使得输入特征之间的相关性降低，并且所有的特征具有相同的方差。

2. 批归一化Batch Normalization

2.1 引言

  深度神经网络难以训练的一个很重要的原因是：在训练过程中，由于前面隐含层的参数发生变化，后年隐含层输入的分布也会随之变化。于是我们只能设置很小的学习率，模型训练速度因而减慢。这种网络中间层在训练过程中输入数据分布发生改变的现象，称作internal covariate shiftinternal\ covariate\ shiftinternal covariate shift。
  如果我们能在每一层输入的时候，先对数据做一个预处理该多好啊！Batch Normalization就是这么做的：在网络每一层输入的时候，插入一个归一化层，然后再进入下一层网络。不过实际的归一化层还是比较麻烦的，它是一个有参数、可学习的网络层，
  运用批归一化的好处：我们可以设置更大的学习率，而且不用再那么小心翼翼初始化参数。批归一化也可以被看作是一种正则化(regularization)，在某些情况下你可以不用做Dropout了。
