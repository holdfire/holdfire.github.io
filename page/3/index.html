3.5 边界框预测Bounding box prediction

YOLO算法：

  • 先把整张图像分成n×nn \times nn×n个网格；
  • 然后对每个网格都生成一个目标检测的标签，如果一个对象在多个网格中，该对象属于其中心所在的那个网格；
  • 需要注意：此时该对象的标签中的目标框参数需要改成以该网格的左上角为原点了。bx,byb_x,b_ybx​,by​仍然在0和1之间，但bh,bwb_h,b_wbh​,bw​可以大于1，这表明待检测物体有部分在该网格外面。

3.6 交并比 Intersection over Union

  交并比IoU被用来衡量目标检测框结果的好坏，通常约定IoU大于0.5，就认为目标检测正确，其计算公式为：

IoU=A∩BA∪BIoU = \frac {A \cap B} {A \cup B} IoU=A∪BA∩B​

3.7 非极大值抑制 Non-Maxiumum Suppression

  模型可能会对同一个对象做出多次检测，非极大值抑制可以清理这些检测结果，这样每个目标只做一次检测。算法原理：

  • （1）对每个网格做出预测，抛弃那些pc≤0.6p_c \leq 0.6pc​≤0.6的低概率边界框；
  • （2）比较每个检测结果的概率pcp_cpc​，找到最大的那个并认为是最有可能的；
  • （3）然后逐一审视剩下的矩形，那些和这个最大边界框有很高的交并比的框会被抑制；循环步骤3和4直至无边界框剩余
  • （4）最后抛弃那些被抑制的框，得到的就是最终的目标检测结果。
    如果要检测多个类别，应该对每个框的每个类别分别进行非极大值抑制。

3.8 Anchor Boxes

  假设我们使用了2种尺寸的anchor box，那么在每个网格中样本标签为y=[pc,bx,by,bh,bw,c1,c2,c3,pc,bx,by,bh,bw,c1,c2,c3]y = [p_c, b_x, b_y,b_h, b_w, c_1,c_2,c_3, p_c, b_x, b_y,b_h, b_w, c_1,c_2,c_3]y=[pc​,bx​,by​,bh​,bw​,c1​,c2​,c3​,pc​,bx​,by​,bh​,bw​,c1​,c2​,c3​]
通常可以选择5-10种不同形状和大小的anchor box。
算法不好处理的情况1：如果一个网格中有3个对象，但是只有2个anchor box
算法不好处理的情况2：同一个网格中含有2个对象，且它们的anchor box尺寸一样

3.10 候选区域 Region Proposal

  R-CNN网络不对每个滑动窗口做卷积，而是选出一些区域做卷积，选取的方法是：先对图像做分割，根据分割边缘找出约2000个候选区域，然后在这些候选区域上放置边界框跑分类。
  Fast R-CNN网络：用卷积实现滑窗来做分类任务，但得到候选区域的聚类步骤仍然很慢
  Faster R-CNN网络：使用卷积神经网络，而不是传统的分割算法来获得候选区域。

1. 简介

  

2. 边缘提取——HED网络

  整体嵌套边缘检测(Holistically-Nested Edge Detection)是一个基于深度卷积神经网络的图像边缘提取算法。其特点是：（1）整体图像训练和预测；（2）多尺度和多层特征学习。该深度模型利用全卷积神经网络，自动学习丰富的分层表示。（基于侧面响应的深层监督指导）

2.1 多尺度深度学习

  多尺度深度学习可以分为四类，即多流学习(multi-stream learning)、跳网(skip-net learning)、多输入单模型以及独立网训练，如下图所示：（a）多流架构；（b）跳网架构；（c）多尺度输入的单一模型；（d）不同网络独立训练；（e）HED，其中添加多个侧面输出。

  在表示和计算方面，前四种方法存在冗余，HED是一个相对简单的变体，能够从多个尺度产生预测。该结构可以被解释为（d）独立网络的整体嵌套版，包括多个侧输出单流深度网络。隐藏层监督可以改善图像分类任务的优化和泛化。如果需要统一输出，多侧向输出可以灵活地添加额外的融合层。

3. 边缘提取——CASENet网络

  CASENet网络将每个边缘像素与多于一个的边缘类相关联，是一种基于ResNet的端到端深度语义边缘学习的架构，同时也是一种跳层(skip-layer)架构，其中顶层卷积层的类别边缘激活共享并与底层特征的同一集融合，用一种多标签损失函数来监督激活的融合。

4. 轮廓提取——DeepEdge

  DeepEdge利用目标相关特征作为轮廓的高级线索检测。设计了一个多级深度网络，由五个卷积层和一个分叉全连接子网络(bifurcated fully-connected subnetwork)组成。

5. 轮廓提取——CEDN

  利用全卷积编码器-解码器网络(CEDN)，端到端的训练数据来自PASCAL VOC中修正的不准确的多边形标注，

2.2 经典的CNN网络

  1998年提出的LeNet-5：输入层–>卷积层–>avg pool层–>卷积层–>avg pool层–>FC-1层–>FC-2层–>输出层。使用sigmoid或tanh激活函数，作者在池化层和面也用了激活函数 。（建议看part 2和part 3）
  2012年提出的AlexNet：输入是224×224×3224 \times 224 \times 3224×224×3的，吴恩达认为227×227×3227 \times 227 \times 3227×227×3更好。输入层–>conv layer–>maxpool layer–>conv layer–>maxpool layer–>conv layer–>maxpool layer–>conv layer–>maxpool layer–>FC-1 layer–>FC-2 layer–>Softmax。一共有60,000,000个参数。原文还有个“局部响应归一化层(Local Response Normalization)”，现在已不使用。
  2015年提出的VGG-16：其简单之处在于卷积层的filter都是3×3,s=1,padding=same3 \times 3, s=1, padding = same3×3,s=1,padding=same的，maxpool层都是2×2,s=22 \times 2, s = 22×2,s=2的。池化层将高度和宽度减半了，经过卷积层时通道数翻倍了。VGG-19和VGG-16的性能不相上下。

2.3残差网络ResNet

  ResNet是由残差块构成，其特点是shortcut或叫做skip connection。假设原有一个plain network：a[l]−>a[l+1]−>a[l+2]a^{[l]} -> a^{[l+1]} ->a^{[l+2]}a[l]−>a[l+1]−>a[l+2]，l+1l+1l+1层的a[l]a^{[l]}a[l]经过线性计算后，加上a[l+1]，再进入a^{[l+1]}，再进入a[l+1]，再进入l+2$层的激活单元。即有:

a[l+2]=g(z[l+2]+a[l])a^{[l+2]} = g(z^{[l+2]} + a^{[l]}) a[l+2]=g(z[l+2]+a[l])

  为什么ResNet能够训练更深的网络？因为残差连接块中，如果a[l]a^{[l]}a[l]什么都不学习，也能保证a[l]=a[l+2]a^{[l]}=a^{[l+2]}a[l]=a[l+2]

2.5 网络中的网络或1×11 \times 11×1卷积核

  1×11 \times 11×1卷积核也被称为网络中的网络，其作用有：可以压缩或者增加通道数目，增加网络结构的非线性能力。

2.6 Inception Net(这名字来自于盗梦空间。。。也叫GoogLeNet)

  作者的出发点是：如果你不确定该使用多大尺寸的卷积核，也不确定是否需要创建卷积层或者池化层，so let’s do them all。由一个层到另一个层，使用分组卷积：包含1×11 \times 11×1,3×33 \times 33×3,5×55 \times 55×5以及maxpool的卷积核的组合。
  当卷积的计算量很大时，在中间插入一个由1×11 \times 11×1卷积核构成的卷积层，其通道数较小，这种做法能够减小计算量。这个卷积层也被叫做bottlenecklayerbottleneck layerbottlenecklayer。
  其他的如Inception V2, V3, V4都是相同的思路，用Inception block。

2.9 迁移学习transfer learning

  使用开源的网络结构和权重参数，对网络最后的几层做修改，将前面层的参数freeze（被freeze的层相当于一个固定的函数），然后用自己的数据去训练网络。通常如果你的数据越多，你需要冻结的层就越少。

2.10 数据增强data augmentation

  镜像mirror，随机裁剪random cropping（并不完美但实用），旋转rotation，剪切变化shearing，local wraping
  颜色偏移color shifting：对某个分量增加或减少一个固定值，PCA color augmentation
  从硬盘中抽取出原图，在内存中做数据增强。

2.11 如何做CNN研究

  当你的数据集很大时，你可以设计很简单的模型，做很少的人工工程；而当你的数据量很少时，采取一定的hand-engineering是一个很好的方式。
Two sources of knoeledge:

  • Labeled data
  • Hand engineering feature/ network architecture/ other components

Tips for doing well on benchmarks/winning competitions:

  • Ensembling: train several networks independently and average their outputs.
  • Multi-crop at test time: run classifier on multiple versions of test image and average results.

Use open source code:

  • Use architectures of networks published in the literature;
  • Use open source implementation if possible;
  • Use pretrained models and fine-tune on your dataset.

4.1 人脸识别

  • Face vertification vs Face recognition
    人脸验证Face vertification：给定一张图片和一个人的名字或ID，验证这个图片是不是这个人。这是一个1：1问题。
    人脸识别Face recognition：给定一个有k个人的数据库，对于输入图片，要求输出该图片是哪一个人，或者返回未识别。这是一个1：k问题

4.2 one-shot learning问题

  • 从每个类别只有一张图片的数据集中学习，然后做检测。
  • 我们要学习一个similarity function，表示两张图片的相似程度；
  • if d(img1,img2)≤τd(img1, img2) \leq \taud(img1,img2)≤τ，那么就认为两张图片是相同的；
  • else if d(img1,img2)>τd(img1, img2) > \taud(img1,img2)>τ，就认为不是相同的。其中τ\tauτ是一个超参数

1.4 padding

  做padding的目的是避免图片边缘的信息丢失了。
  选择padding的个数，通常有两种选择：valid和same。valid表示没有填充；same表示：“Pad so that output size is the same as the input size.”
  通常情况下，卷积核的尺寸是奇数。如果卷积核尺寸是偶数，stride等于1，pandding为same，这时候只能设置不对称的padding了。

1.8 简单卷积网络示例

  一般随着网络的加深，中间层的尺寸刚开始保持一致，后面不断变小；通道数不断增加。
  卷积神经网络中通常有3种layer：卷积层convolution layer，池化层pooling layer，全连接层fully connected。

1.9 池化层

  池化层没有参数要学习，池化层的输出通道数和输入通道数一样。
  池化层的超参数主要是filter size和stride，二者通常均被设置为2，很少设置padding。
  Max pooling层的作用：如果过滤器filter提取到了某个特征，那么保留其最大值。
  Average polling层：使用较少。

1.10 卷积神经网络示例

  在计算卷积神经网络的层数时有：有人把卷积层和池化层作为一层，也有人把卷积层和池化层分开算作两层。
  吴恩达老师把卷积操作和池化操作合起来当作一层。
  卷积神经网络的activation size（每一层的变量个数）通常是在下降，但不能下降太快。
  选择超参数时，尽量参考别人的文章或者网络结构。

1.11 为什么使用卷积

  卷积层和全连接层相比的优势：通过参数共享和稀疏连接减少参数，防止过拟合。
  Parameter sharing: A feature detector(such as a vertical edge detector) that is useful in one part of the image is probably useful in another part of the image.
  Sparsity of connections: In early layer, each output value depends only on a small number of inputs.
